Bienvenido

Este sitio reúne notas y ejemplos cortos para Álgebra y Funciones. Las fórmulas se renderizan con MathJax (LaTeX en la web).

Ejemplo LaTeX

$$\sin^2(x)+\cos^2(x)=1$$

Puedes escribir ecuaciones con $$ ... $$.

Cómo usar el sitio

  • Entra a cualquier tema desde el menú.
  • Explora la sección Quizzes para actividades interactivas.
  • Edita los archivos .html para ampliar contenido.
  • Luego solo haces Commit y la web se actualiza.
